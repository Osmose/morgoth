Morgoth
========
Morgoth is a service that helps Firefox developers manage the deployment
pipeline for `System Add-ons`_. It acts as both a tool for tracking the status
of system add-ons and for controlling `Balrog`_ to deploy them.

.. _System Add-ons: https://wiki.mozilla.org/Firefox/Go_Faster
.. _Balrog: https://wiki.mozilla.org/Balrog
